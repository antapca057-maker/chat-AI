# NextChat 0.0.20

Лёгкое чат-приложение с ИИ для Android, оптимизированное для слабых устройств.

## 📦 Характеристики

- **Размер**: ~39,5 МБ
- **Совместимость**: Android 7.0+ (API 21+)
- **Оптимизация**: Работает на слабых устройствах (Redmi 4X)
- **Функционал**:
  - 🎙️ Голосовой ввод (speech-to-text)
  - 🔊 Голосовой вывод (text-to-speech)
  - 💬 Поддержка нескольких диалогов
  - 🖼️ Обмен изображениями и скринами
  - 💾 Сохранение истории локально
  - 🌙 Тёмная тема по умолчанию
  - 📝 Простой и приятный интерфейс

## 🚀 Установка

### Требования

- Flutter SDK 3.0+
- Android SDK (API 21+)
- 500MB свободного места

### Быстрый старт

1. Установите Flutter: https://flutter.dev/docs/get-started/install

2. Клонируйте проект:
```bash
cd nextchat
```

3. Получите зависимости:
```bash
flutter pub get
```

4. Запустите на устройстве:
```bash
flutter run
```

5. Для сборки APK:
```bash
flutter build apk --release
```

## 📁 Структура проекта

```
lib/
├── main.dart              # Точка входа
├── models/
│   └── chat_model.dart    # Модели данных
├── providers/
│   └── chat_provider.dart # Управление состоянием
├── screens/
│   ├── home_screen.dart   # Главный экран
│   └── chat_screen.dart   # Экран чата
└── widgets/
    └── message_bubble.dart # Компонент сообщения
```

## 🔧 Использованные пакеты

- **hive_flutter** - локальное хранилище
- **provider** - управление состоянием
- **speech_to_text** - голосовой ввод
- **flutter_tts** - голосовой вывод
- **image_picker** - работа с изображениями
